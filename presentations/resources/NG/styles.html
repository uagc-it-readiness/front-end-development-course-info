Component Styles

Styling components in Angular is different than traditional styling.

Inline Syntax


@Component({
  selector: 'app-comp',
  template: `
    
Hello Angular

  `,
  styles: [`
    h1 { font-weight: normal; }
    h2 { font-weight: bold; }
   `]
})
export class HeroAppComponent {
/* . . . */
}

External Syntax


@Component({
  selector: 'app-comp',
  template: `
    
Hello Angular

  `,
    styleUrls: ['app.component.css']
})
export class HeroAppComponent {
/* . . . */
}

CSS Encapsulation


@Component({
  selector: 'app-comp',
  template: `
    
Hello Angular

  `,
  styles: [`
    h1 { font-weight: normal; }
    h2 { font-weight: bold; }
   `]
})
export class HeroAppComponent {
/* . . . */
}
