Filtering jQuery Further

We've leveraged the .filter() method to shrink a collection of jQuery objects. These hard handy, but don't do much if you are only wanting to get a single element.

.eq()

The .eq() method retrieves the jQuery object at the specific index that is passed in. 

            
         var lis = $('li');
        var first = lis.eq(0);
        /////////

        //Take immediate action on the first one.
        $('li').eq(0).hide();

.get()

.get() acts similar to .eq(), BUT it returns a DOM object instead of a jQuery object. 

        
        var lis = $('input');
        var first = lis.get(0);
        console.log(first.value);
