Margins

CSS Margins add spacing around an element.

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Padding

CSS Padding add spacing inside an element.

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Shorthand

  • 5px applies 5 pixels to all 4 sides.
  • 5px 10px applies 5px to top and bottom and applies 10px to left and right.
  • 5px 10px 15px 20px
    • 5px to top
    • 10px to right
    • 15px to bottom
    • 20px to left

Margin Shorthand


.fig10 {
    /* Applies 5px to all 4 sides */
    margin: 5px;
}
.fig11 {
    /* Applies 5px to all 4 sides */
    margin: 5px 5px 5px 5px;
}
.fig12{
    /*Applies 10px to top and bottom
      and 30px left and right*/
    margin:10px 30px;
}

Height & Width

Sets the specific height and width of a block element

  • height
    • min-height
    • max-height
  • width
    • min-width
    • max-width

Examples


.fig15{
    height:100px;
    width:250px;
}
/*Applying this to a span tag*/
.fig16{
    height:100px;
    width:100px;
}
.fig17{
    max-width:100px;

Display

The CSS display property is used to change the default rendering style of an element.

  • inline - span, em, strong
  • block - div, p, h1-h6
  • none - hides the element
  • inline-block - Renders block, but stays in a single line.

You can change any element to display any type.

Set inline to block, but not block to inline.

List Style

Changes the default styling of UL and OL lists

  • list-style-type - changes the icon/value used in the list
  • list-style-position - changes the position of the list. Accepts inside and outside
  • list-style-img - changes bullets to image. Accepts none or a valid URL
  • Short handlist-style: type position img

Grouping Selectors

You can leverage grouping selectors to share styles between multiple selectors. This can be done by adding a comma between each selector. 


    h1, h2, .header, #site-header{
        color:#ff0000;
    }

ls

Changing a Directory

cd

MyForks/   presentations/  'year 1 slides'/

$ cd MyForks/

project1/ project2/

Directory Shortcuts

  • . - Current Directory
  • .. - Parent Directory
  • ~ - User Home - C:\Users\jisajmb
  • / - Root

Displaying your current path.

Got lost in your console? Use the print working directory command to get a full path of your current location

pwd


$ cd ~
$ pwd
c/users/jisajmb

Creating a directory/folder

You can create a new folder by using the make directory command.

mkdir


#list directory
$ ls

MyForks/   presentations/  'year 1 slides'/

$ mkdir my-work 
$ ls

my-work/  MyForks/   presentations/  'year 1 slides'/

Make a file via command line

touch


#list directory
$ ls

my-work/ MyForks/   presentations/  'year 1 slides'/

$ cd my-work 
$ ls


$ touch index.html
$ ls

index.html

The Clear Command

You can clear the console by simply typing clear and hitting enter.
