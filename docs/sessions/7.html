Backgrounds

Changes the background appearance of the element.

  • background-color
  • background-image
  • background-repeat repeats the background image.
    • repeat-x repeats the image across the x axis
    • repeat-y repeats the image across the y axis
    • no-repeat makes the image only appear once

See the Pen opRXNZ by Jon (@Middaugh) on CodePen.

Backgrounds Continued

  • background-position Accepts two values. First the x and then the y. You specify these by using these values.
    • left|right|center  top|center|bottom
    • 1px 1px; - This would be 1px off the top and 1px off the left
    • x% y%

Examples


.item{
    background-color: #ff0000;
    background-color: blue;
    
    background-image: url(http://via.placeholder.com/350x150);
    background-image: url(../images/me.jpg);
    
    background-repeat: repeat-x;
    background-repeat: no-repeat;    
    
    background-position: center center;
    background-position: 10px 10px;
    
}

Borders

This CSS property defines the borders around an element.

  • border-width accepts pixel value
  • border-style
    • solid the most common
    • dashed
    • dotted
    • Many others
  • color accepts any valid color
border width also accepts thin medium and thick, but px is the most desired

Examples


    .fig1{
        border-color: red;
        border-style: solid;
        border-width:5px;
    }
    .fig2{
        border-color: navy;
        border-style: dotted;
        border-width:10px;
    }
Red border that is 5px thick.

    .fig3 {
        border: 5px solid red;
    }
    .fig4 {
        border:10px dotted navy;

Margins

CSS Margins add spacing around an element.

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Padding

CSS Padding add spacing inside an element.

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Shorthand

  • 5px applies 5 pixels to all 4 sides.
  • 5px 10px applies 5px to top and bottom and applies 10px to left and right.
  • 5px 10px 15px 20px
    • 5px to top
    • 10px to right
    • 15px to bottom
    • 20px to left

Margin Shorthand


.fig10 {
    /* Applies 5px to all 4 sides */
    margin: 5px;
}
.fig11 {
    /* Applies 5px to all 4 sides */
    margin: 5px 5px 5px 5px;
}
.fig12{
    /*Applies 10px to top and bottom
      and 30px left and right*/
    margin:10px 30px;
}

Display

The CSS display property is used to change the default rendering style of an element.

  • inline - span, em, strong
  • block - div, p, h1-h6
  • none - hides the element
  • inline-block - Renders block, but stays in a single line.

You can change any element to display any type.

Set inline to block, but not block to inline.

List Style

Changes the default styling of UL and OL lists

  • list-style-type - changes the icon/value used in the list
  • list-style-position - changes the position of the list. Accepts inside and outside
  • list-style-img - changes bullets to image. Accepts none or a valid URL
  • Short handlist-style: type position img

Grouping Selectors

You can leverage grouping selectors to share styles between multiple selectors. This can be done by adding a comma between each selector. 


    h1, h2, .header, #site-header{
        color:#ff0000;
    }
